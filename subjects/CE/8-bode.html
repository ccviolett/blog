                          # 绘制伯德图

                          对数幅频特性曲线由各典型环节对数幅频特性叠加而成，相频特性曲线由各典型环节相频特性叠加而成。

                          1. 将系统传递函数写成标准形式（各环节常数项为 1），并求出其频率特性
                          2. 确定各典型环节的转角频率，并由小到大按其顺序标在横坐标上
                          3. 计算20lg⁡K20\lg K20lgK，找到横坐标ω=1\omega = 1ω=1，纵坐标为20lg⁡K20\lg K20lgK的点
                          4. 过点开始绘制折线，如有积分环节/二重积分环节，过该点做斜率为−20dB/dec-20 dB/dec−20dB/dec/−40dB/dec-40dB/dec−40dB/dec的斜线，以后每遇到一个转角频率便改变一次频率：
                            1. 如遇惯性环节的转角频率，斜率增加−20dB/dec-20dB/dec−20dB/dec
                            2. 如遇一阶微分环节的转角频率，斜率增加+20dB/dec+20dB/dec+20dB/dec
                            3. 如遇振荡环节的转角频率，斜率增加−40dB/dec-40dB/dec−40dB/dec
                            4. 如遇二阶微分环节的转角频率，斜率增加+40dB/dec+40dB/dec+40dB/dec
                          5. 根据误差修正曲线对渐近线进行修正。
                          例题1（无积分环节的伯德图绘制）

                          作传递函数为G(s)=24(0.25s+0.5)(5s+2)(0.05s+2)G(s) = \frac{24(0.25s+0.5)}{(5s + 2)(0.05s + 2)}G(s)=(5s+2)(0.05s+2)24(0.25s+0.5)​的系统的 Bode 图。

                          解：化为标准形式G(s)=24(0.25s+0.5)(5s+2)(0.05s+2)G(s) = \frac{24(0.25s+0.5)}{(5s + 2)(0.05s + 2)}G(s)=(5s+2)(0.05s+2)24(0.25s+0.5)​，包含一个比例环节、一个一阶微分环节，两个惯性环节串联组成。

                          已知频率特性为G(jω)=3(0.5jω+1)(2.5jω+1)(0.025jω+1)G(j\omega) = \frac{3(0.5j\omega + 1)}{(2.5j\omega + 1)(0.025j\omega + 1)}G(jω)=(2.5jω+1)(0.025jω+1)3(0.5jω+1)​，求得各环节的转角频率：

                          # 最小相位系统

                          系统开环传递函数在 S 右边平面上既无极点、又无零点的系统，称为最小相位系统；否则，为非最小相位系统。

                          最小相位系统的相频特性和幅频特性是一一对应的，大多数系统都是最小相位系统。

                          有延迟环节的系统属于非最小相位系统。

                          例题3（最小相位系统的传递函数）

                          如图均是最小相位系统的开环对数幅频特性曲线，试写出开环传递函数。
