                        极坐标图是反映频率特性的几何表示。

                        它是当角频率从 ω\omegaω 从 000 逐渐增加到 +∞+\infty+∞ 时，矢量 ∣G(jω)∣⋅ejφ(ω)|G(j\omega)|\cdot e^{j\varphi(\omega)}∣G(jω)∣⋅ejφ(ω) 的端点在 [G(jω)][G(j\omega)][G(jω)] 复平面（频域）上相对应的轨迹。

                        逆时针为正

                        # 典型环节的 Nyquist 图

                        提示

                        例：延时环节e−3se^{-3s}e−3s的相频特性φ(ω)=3−ω\varphi(\omega) = 3-\omegaφ(ω)=3−ω。

                        环节备注
                        比例环节理想的放大环节能够无失真和无滞后地复现输入信号
                        积分环节积分环节对正弦信号有90∘90^{\circ}90∘的滞后作用。
                        微分环节微分环节对正弦输入有90∘90^{\circ}90∘的超前作用。
                        延迟环节相角0∘∼−∞∘0^{\circ} \sim -\infty^{\circ}0∘∼−∞∘，与实轴和虚轴有无穷多交点

                        # 二阶振荡环节的 Nyquist 图

                        已知二阶振荡环节的传递函数为G(s)=1T2s2+2ζTs+1G(s) = \frac{1}{T^{2}s^{2} + 2\zeta Ts + 1}G(s)=T2s2+2ζTs+11​，则其频率特性为G(jω)=1T2(jω)2+2ζT(jω)+1G(j\omega) = \frac{1}{T^{2}(j\omega)^{2} + 2\zeta T(j\omega) + 1}G(jω)=T2(jω)2+2ζT(jω)+11​，可得：

                        ∣G(jω)∣=1(1−T2ω2)2+(2ζTω)2∠G(jω)={−arctan⁡2ζTω1−T2ω2ω≤1T−(π−arctan⁡2ζTωT2ω2−1)ω>1T \begin{align} |G(j\omega)| & = \frac{1}{\sqrt{ (1 - T^{2}\omega^{2})^{2} + (2\zeta T\omega)^{2} }} \\ \angle G(j\omega) & = \begin{cases} -\arctan \frac{2\zeta T\omega}{1 - T^{2}\omega^{2}} & \omega \leq \frac{1}{T} \\ -\left( \pi - \arctan \frac{2\zeta T\omega}{T^{2}\omega^{2} - 1} \right) & \omega > \frac{1}{T} \end{cases} \end{align} ∣G(jω)∣∠G(jω)​=(1−T2ω2)2+(2ζTω)2​1​={−arctan1−T2ω22ζTω​−(π−arctanT2ω2−12ζTω​)​ω≤T1​ω>T1​​​​

                        其 Nyquist 图如下：

                        相角0∘∼−180∘0^{\circ} \sim -180^{\circ}0∘∼−180∘，与负虚轴有交点。令∠G(jω)=−90∘\angle G(j\omega) = -90^{\circ}∠G(jω)=−90∘，得ω=1T=ωn\omega = \frac{1}{T} = \omega_nω=T1​=ωn​，G(jωn)=12ζ∠−90∘G(j \omega_n) = \frac{1}{2\zeta}\angle -90^{\circ}G(jωn​)=2ζ1​∠−90∘为负虚轴交点。

                        # 绘制乃氏图

                        1. 写出幅频特性和相频特性表达式
                        2. 分别求出ω\omegaω趋近与000和∞\infty∞时的频率特性
                        3. 求乃氏图与实轴的交点，令Im[G(jω)]=0Im[G(j\omega)] = 0Im[G(jω)]=0或令∠G(jω)=n⋅180∘\angle G(j\omega) = n \cdot 180^{\circ}∠G(jω)=n⋅180∘求出
                        4. 求乃氏图与虚轴的交点，令Re[G(jω)]=0Re[G(j\omega)] = 0Re[G(jω)]=0或令∠G(jω)=n⋅90∘\angle G(j\omega) = n \cdot 90^{\circ}∠G(jω)=n⋅90∘求出
                        5. 必要时画出乃氏图中间几点
                        6. 勾画出大致曲线
                        例题1（乃氏图的绘制）

                        试画出G(s)=1(s+1)(2s+1)G(s) = \frac{1}{(s + 1)(2s + 1)}G(s)=(s+1)(2s+1)1​的乃氏图

                        解：频率特性G(jω)=1(jω+1)(2jω+1)G(j\omega) = \frac{1}{(j\omega + 1)(2 j\omega + 1)}G(jω)=(jω+1)(2jω+1)1​故

                        {∣G(jω)∣=1ω2+1(2ω)2+1∠G(jω)=−arctan⁡(ω)−arctan⁡(2ω) \begin{cases} |G(j\omega)| & = \frac{1}{\sqrt{ \omega^{2}+1 }\sqrt{ (2\omega)^{2} + 1 }} \\ \angle G(j\omega) & = -\arctan(\omega) - \arctan(2\omega) \end{cases} {∣G(jω)∣∠G(jω)​=ω2+1​(2ω)2+1​1​=−arctan(ω)−arctan(2ω)​

                        当ω=0\omega = 0ω=0时，G(j0)=1∠0∘G(j 0) = 1\angle 0^{\circ}G(j0)=1∠0∘；当ω→∞\omega \to \inftyω→∞时，G(j∞)=0∠−180∘G(j\infty) = 0 \angle -180^{\circ}G(j∞)=0∠−180∘。

                        其相角范围从−0∘∼−180∘-0^{\circ} \sim -180^{\circ}−0∘∼−180∘，因此必有与负虚轴的交点

                        令∠G(jω)=−arctan⁡(ω)−arctan⁡(2ω)=−90∘\angle G(j\omega) = -\arctan(\omega) - \arctan(2\omega) = -90^{\circ}∠G(jω)=−arctan(ω)−arctan(2ω)=−90∘，即arctan⁡(ω)+arctan⁡(2ω)=90∘\arctan(\omega) + \arctan(2\omega) = 90^{\circ}arctan(ω)+arctan(2ω)=90∘

                        两角互余，正切互为倒数，故两边取正切，得2ω=1ω2\omega = \frac{1}{\omega}2ω=ω1​，即ω=12=22\omega = \sqrt{\frac{1}{2}} = \frac{\sqrt{2}}{2}ω=21​​=22​​

                        例题2（绘制乃氏图）

                        画出G(s)=1s(s+1)(2s+1)G(s) = \frac{1}{s(s + 1)(2s + 1)}G(s)=s(s+1)(2s+1)1​的乃氏图。

                        解：G(jω)=1jω(jω+1)(2jω+1)G(j\omega) = \frac{1}{j\omega(j\omega + 1)(2j\omega + 1)}G(jω)=jω(jω+1)(2jω+1)1​ ，故

                        {∣G(jω)∣=1ωω2+1⋅(2ω)2+1∠G(jω)=−90∘−arctan⁡(ω)−arctan⁡(2ω) \begin{cases} |G(j\omega)| & = \frac{1}{\omega \sqrt{ \omega^{2} + 1 }\cdot \sqrt{ (2\omega)^{2} + 1 }} \\ \angle G(j\omega) & = -90^{\circ} - \arctan(\omega) - \arctan(2\omega) \end{cases} {∣G(jω)∣∠G(jω)​=ωω2+1​⋅(2ω)2+1​1​=−90∘−arctan(ω)−arctan(2ω)​

                        当ω=0\omega = 0ω=0时，G(jω)=∞∠−90∘G(j\omega) = \infty \angle -90^{\circ}G(jω)=∞∠−90∘；当ω→∞\omega \to \inftyω→∞时，G(jω)=0∠−270∘G(j\omega) = 0 \angle -270^{\circ}G(jω)=0∠−270∘

                        其相角范围从−90∘∼−270∘-90^{\circ}\sim-270^{\circ}−90∘∼−270∘，因此必有与负实轴的交点

                        令∠G(jω)=−90∘−arctan⁡(ω)−arctan⁡(2ω)=−180∘\angle G(j\omega) = -90^{\circ} - \arctan(\omega) - \arctan(2\omega) = -180^{\circ}∠G(jω)=−90∘−arctan(ω)−arctan(2ω)=−180∘，即arctan⁡(2ω)=90∘−arctan⁡(ω)\arctan(2\omega) = 90^{\circ} - \arctan(\omega)arctan(2ω)=90∘−arctan(ω)

                        两角互余，正切互为倒数，两边取正切，得得2ω=1ω2\omega = \frac{1}{\omega}2ω=ω1​，即ω=12=22\omega = \sqrt{\frac{1}{2}} = \frac{\sqrt{2}}{2}ω=21​​=22​​
